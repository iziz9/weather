# Weather
현재 시간과 위치, 날씨 정보를 표시해주는 위젯입니다.

## 배포 링크

[✨ 배포 링크](https://cinema-paradiso-green.vercel.app/)

## 설치 및 실행방법

```shell
$ git clone https://github.com/iziz9/weather

$ cd weather

$ yarn install

$ yarn dev
```

## 사용API 
- 날씨 : https://openweathermap.org/
- 지오코딩 : https://opencagedata.com/
